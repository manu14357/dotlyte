## Installation

```toml
# Cargo.toml
[dependencies]
dotlyte = "0.1"
```

Or via CLI:

```bash
cargo add dotlyte
```

### Requirements

- Rust **edition 2021** (1.56+)
- Default features include YAML and TOML support
- Optional `encryption` feature for AES-256-GCM

### Feature Flags

| Feature | Dependencies | Default |
|---|---|---|
| `yaml` | `serde_yaml` | Yes |
| `toml-support` | `toml` | Yes |
| `encryption` | `aes-gcm`, `scrypt`, `base64`, `hex` | No |

```toml
# Enable encryption
dotlyte = { version = "0.1", features = ["encryption"] }
```

## Quick Start

```rust
use dotlyte::load;

fn main() -> dotlyte::Result<()> {
    let config = load(None)?;

    let host: String = config.get("database.host")
        .unwrap_or("localhost".into());
    let port: i64 = config.get("database.port")
        .unwrap_or(5432);

    println!("DB: {}:{}", host, port);
    Ok(())
}
```

## API Reference

### `load(options: Option<LoadOptions>) -> Result<Config>`

```rust
use dotlyte::{load, LoadOptions};

// Basic (zero-config)
let config = load(None)?;

// With options
let config = load(Some(LoadOptions {
    files: Some(vec![".env".into(), "config.yaml".into()]),
    prefix: Some("APP_".into()),
    env: Some("production".into()),
    ..Default::default()
}))?;
```

### `LoadOptions`

```rust
#[derive(Debug, Default, Clone)]
pub struct LoadOptions {
    pub files: Option<Vec<String>>,                        // Explicit files to load
    pub prefix: Option<String>,                            // Env var prefix to strip
    pub defaults: Option<serde_json::Map<String, Value>>,  // Default values (lowest priority)
    pub sources: Option<Vec<String>>,                      // Source order: "env","dotenv","yaml","json","toml","defaults"
    pub env: Option<String>,                               // Environment name
    pub schema: Option<DotlyteSchema>,                     // Schema for validation
    pub strict: bool,                                      // Reject unknown keys
    pub interpolate_vars: bool,                            // Enable ${VAR} interpolation
    pub overrides: Option<serde_json::Map<String, Value>>, // Highest priority values
    pub debug: bool,                                       // Debug logging
    pub find_up: bool,                                     // Walk up directories for config
    pub root_markers: Option<Vec<String>>,                 // Stop markers for find_up
    pub cwd: Option<String>,                               // Working directory
    pub allow_all_env_vars: bool,                          // Skip env var filtering
    pub watch: bool,                                       // Enable file watching
    pub debounce_ms: Option<u64>,                          // Watcher debounce interval
}
```

### `Config`

#### `get<T: FromValue>(key) -> Option<T>`

Generic typed access. `FromValue` is implemented for `String`, `i64`, `f64`, `bool`:

```rust
let host: Option<String> = config.get("server.host");
let port: Option<i64> = config.get("server.port");
let debug: Option<bool> = config.get("debug");
```

#### `get_value(key) -> Option<&Value>`

Get the raw `serde_json::Value`:

```rust
if let Some(val) = config.get_value("nested.key") {
    println!("{}", val);
}
```

#### `get_str(key) -> Option<&str>`

Direct string access without allocation:

```rust
if let Some(host) = config.get_str("server.host") {
    println!("Host: {}", host);
}
```

#### `require(key) -> Result<&Value>`

Returns the value or `DotlyteError::MissingKey`:

```rust
let db_url = config.require("database_url")?;
```

#### `require_keys(keys) -> Result<()>`

Validate all keys exist:

```rust
config.require_keys(&["db.host", "db.port", "db.name"])?;
```

#### `has(key) -> bool`

```rust
if config.has("redis.host") {
    let host = config.require("redis.host")?;
}
```

#### `scope(prefix) -> Config`

Create a scoped sub-config:

```rust
let db = config.scope("database");
let host: Option<String> = db.get("host"); // same as config.get("database.host")
```

#### Serialization & Export

```rust
config.keys();           // Vec<String> — top-level keys
config.to_map();         // &Map<String, Value> — raw data reference
config.to_value();       // Value — deep clone
config.to_flat_map();    // HashMap<String, String> — flattened
config.to_map_redacted(); // Map<String, Value> — sensitive values → "[REDACTED]"
config.to_json();        // String — JSON with redaction
config.write_to("out.json")?; // Write to file
```

#### Schema Validation

```rust
use dotlyte::{DotlyteSchema, SchemaRule};

let violations = config.validate();    // Vec<SchemaViolation>
config.assert_valid()?;                // returns Result<()>
```

### `FromValue` Trait

Enables generic `config.get<T>()`:

```rust
pub trait FromValue: Sized {
    fn from_value(value: &Value) -> Option<Self>;
}
// Implemented for: String, i64, f64, bool
```

### Encryption (AES-256-GCM)

Requires the `encryption` feature:

```rust
use dotlyte::{generate_key, encrypt_value, decrypt_value, is_encrypted};

let key = generate_key()?;

let encrypted = encrypt_value("s3cret", &key)?;  // "ENC[aes-256-gcm,...]"
let plain = decrypt_value(&encrypted, &key)?;      // "s3cret"

assert!(is_encrypted(&encrypted));
```

### Sensitive Value Masking

```rust
use dotlyte::{REDACTED, build_sensitive_set, redact_map, format_redacted};

let redacted = config.to_map_redacted();
// {"db.host": "localhost", "db.password": "[REDACTED]"}
```

### Custom Source Plugin

```rust
use dotlyte::{Source, Result};
use serde_json::{Map, Value};

struct VaultSource;

impl Source for VaultSource {
    fn name(&self) -> &str { "vault" }
    fn load(&self) -> Result<Map<String, Value>> {
        let mut m = Map::new();
        m.insert("secret".into(), Value::String("from-vault".into()));
        Ok(m)
    }
}
```

### Error Types

```rust
#[derive(Debug, thiserror::Error)]
pub enum DotlyteError {
    MissingKey { key: String, sources_checked: Vec<String> },
    ParseError { file: String, message: String },
    FileError { file: String, message: String },
    ValidationError { violations: Vec<SchemaViolation> },
    InterpolationError { variable: String, message: String },
    DecryptionError { message: String },
    Io(#[from] std::io::Error),
}
```

Pattern matching:

```rust
use dotlyte::{load, DotlyteError};

match load(None) {
    Ok(config) => { /* use config */ }
    Err(DotlyteError::FileError { file, message }) => {
        eprintln!("File error: {} — {}", file, message);
    }
    Err(DotlyteError::MissingKey { key, sources_checked }) => {
        eprintln!("Missing: {} (checked: {:?})", key, sources_checked);
    }
    Err(e) => eprintln!("Error: {}", e),
}
```

### Schema Types

```rust
pub type DotlyteSchema = std::collections::HashMap<String, SchemaRule>;

pub struct SchemaRule {
    pub rule_type: Option<String>,    // "string","number","boolean","array","object"
    pub required: bool,
    pub format: Option<String>,       // "url","email","ip","port","uuid","date"
    pub pattern: Option<String>,      // regex
    pub enum_values: Option<Vec<Value>>,
    pub min: Option<f64>,
    pub max: Option<f64>,
    pub default: Option<Value>,
    pub sensitive: bool,
    pub doc: Option<String>,
}

pub struct SchemaViolation {
    pub key: String,
    pub message: String,
    pub rule: String,
}
```

## Patterns

### Actix-Web

```rust
use actix_web::{web, App, HttpServer};
use dotlyte::load;

#[actix_web::main]
async fn main() -> std::io::Result<()> {
    let config = load(None).expect("Failed to load config");
    let port: i64 = config.get("port").unwrap_or(8080);

    HttpServer::new(|| {
        App::new()
            .route("/", web::get().to(|| async { "Hello, DOTLYTE!" }))
    })
    .bind(("0.0.0.0", port as u16))?
    .run()
    .await
}
```

### Axum

```rust
use axum::{routing::get, Router};
use dotlyte::load;

#[tokio::main]
async fn main() {
    let config = load(None).expect("Failed to load config");
    let port: i64 = config.get("port").unwrap_or(3000);

    let app = Router::new().route("/", get(|| async { "Hello!" }));
    let addr = format!("0.0.0.0:{}", port);
    let listener = tokio::net::TcpListener::bind(&addr).await.unwrap();
    axum::serve(listener, app).await.unwrap();
}
```

### Error Handling with `?` and anyhow

```rust
use anyhow::Result;
use dotlyte::load;

fn setup() -> Result<()> {
    let config = load(None)?;
    let _db_url = config.require("database_url")?;
    Ok(())
}
```

## What's New in v0.1.2

### Typed Config

```rust
use dotlyte::typed::{create_typed_config, FieldDescriptor};

let schema = HashMap::from([
    ("PORT".into(), FieldDescriptor {
        field_type: "integer".into(),
        required: true,
        min: Some(1.0), max: Some(65535.0),
        ..Default::default()
    }),
]);

let config = create_typed_config(&schema, false)?;
```

### Boundary Config

```rust
use dotlyte::boundary::create_boundary_config;

let config = create_boundary_config(
    &["DATABASE_URL", "API_SECRET"],
    &["APP_NAME", "THEME"],
)?;

let server = config.server();
let client = config.client(); // Cannot access server keys
```

### Enhanced Encryption & Masking

```rust
use dotlyte::encryption::Encryption;
use dotlyte::masking::Masking;

let enc = Encryption::new("my-secret-key-32-chars-long!!!!!")?;
let encrypted = enc.encrypt("secret")?;
let decrypted = enc.decrypt(&encrypted)?;

let m = Masking::new(&["API_KEY"]);
let result = m.mask(&config_map);
```
