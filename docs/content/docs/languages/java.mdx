## Quick Start

```java
import dev.dotlyte.Dotlyte;
import dev.dotlyte.Config;

public class App {
    public static void main(String[] args) {
        Config config = Dotlyte.load();

        Object host = config.get("database.host", "localhost");
        int port = config.getInt("database.port", 5432);

        System.out.printf("DB: %s:%d%n", host, port);
    }
}
```

## API Reference

### `Dotlyte.load(options?)`

```java
import dev.dotlyte.Dotlyte;
import dev.dotlyte.Config;
import dev.dotlyte.LoadOptions;

// Basic
Config config = Dotlyte.load();

// With options
Config config = Dotlyte.load(LoadOptions.builder()
    .files(List.of(".env", "config.yaml"))
    .prefix("APP_")
    .defaults(Map.of("port", 3000))
    .env("production")
    .build());
```

### `LoadOptions`

Built via `LoadOptions.builder()`:

```java
LoadOptions options = LoadOptions.builder()
    .files(List.of(".env", "settings.yaml"))
    .prefix("MYAPP_")
    .defaults(Map.of("log_level", "info"))
    .sources(List.of("env", "dotenv", "yaml"))
    .env("staging")
    .schema(schema)
    .strict(true)
    .interpolateVars(true)
    .overrides(Map.of("key", "forced_value"))
    .debug(true)
    .findUp(true)
    .rootMarkers(List.of(".git", "build.gradle.kts"))
    .cwd("/path/to/project")
    .allowAllEnvVars(false)
    .watch(true)
    .debounceMs(300)
    .build();
```

| Field | Type | Default | Description |
|---|---|---|---|
| `files` | `List<String>` | — | Explicit config file paths |
| `prefix` | `String` | — | Env var prefix to strip |
| `defaults` | `Map<String, Object>` | — | Default values (lowest priority) |
| `sources` | `List<String>` | — | Source order: `"env"`, `"dotenv"`, `"yaml"`, `"json"`, `"toml"`, `"defaults"` |
| `env` | `String` | — | Environment name |
| `schema` | `Map<String, SchemaRule>` | — | Validation rules |
| `strict` | `boolean` | `false` | Reject unknown keys |
| `interpolateVars` | `boolean` | `true` | Enable `${VAR}` interpolation |
| `overrides` | `Map<String, Object>` | — | Highest priority values |
| `debug` | `boolean` | `false` | Debug logging |
| `findUp` | `boolean` | `false` | Walk up directories |
| `rootMarkers` | `List<String>` | — | Project root markers |
| `cwd` | `String` | — | Working directory |
| `allowAllEnvVars` | `boolean` | `false` | Disable env var filtering |
| `watch` | `boolean` | `false` | Enable file watching |
| `debounceMs` | `int` | `300` | Watcher debounce (ms) |

### `Config`

#### `get(String key)` / `get(String key, Object defaultValue)`

Returns `Object` (not `String`):

```java
Object host = config.get("server.host", "0.0.0.0");
Object port = config.get("server.port", 8080);
Object missing = config.get("nope"); // null
```

#### Typed Getters

```java
String host = config.getString("server.host", "0.0.0.0");
int port = config.getInt("server.port", 8080);
boolean debug = config.getBoolean("debug", false);
```

#### `require(String key)`

Returns `Object` or throws `MissingKeyException`:

```java
import dev.dotlyte.DotlyteException.MissingKeyException;

try {
    Object dbUrl = config.require("database_url");
} catch (MissingKeyException e) {
    System.err.println("Missing: " + e.getKey());
    System.err.println("Checked: " + e.getSourcesChecked());
}
```

#### `requireKeys(String... keys)`

Validate multiple keys at startup:

```java
config.requireKeys("db.host", "db.port", "db.name");
```

#### `has(String key)`

```java
if (config.has("redis.host")) {
    setupRedis(config);
}
```

#### `scope(String prefix)`

Create a scoped sub-config:

```java
Config db = config.scope("database");
Object host = db.get("host"); // same as config.get("database.host")
```

#### Serialization & Export

```java
config.keys();           // Set<String> — top-level keys
config.toFlatKeys();     // Set<String> — all flattened dot-notation keys
config.toFlatMap();      // Map<String, String> — flat key→string map
config.toMap();          // Map<String, Object> — deep nested map
config.toMapRedacted();  // Map<String, Object> — sensitive values → "[REDACTED]"
config.toJson();         // String — JSON representation
config.writeTo(Path.of("output.env")); // Writes in dotenv format
```

#### Schema Validation

```java
import dev.dotlyte.SchemaRule;

Map<String, SchemaRule> schema = Map.of(
    "port", new SchemaRule("integer", true, null, null, null, 1.0, 65535.0, null, false, null),
    "host", new SchemaRule("string", true, null, null, null, null, null, null, false, null)
);

List<SchemaViolation> violations = config.validate();
config.assertValid(); // throws ValidationException
```

#### File Watching (Hot-Reload)

When `watch(true)` is set in LoadOptions, the Config includes a `ConfigWatcher`:

```java
Config config = Dotlyte.load(LoadOptions.builder()
    .watch(true)
    .debounceMs(200)
    .build());

// ConfigWatcher runs on a background thread
// Config refreshes automatically when files change
```

### Variable Interpolation

```properties
# In .env:
BASE_URL=https://api.example.com
API_ENDPOINT=${BASE_URL}/v1
GREETING=${NAME:-World}
REQUIRED=${SECRET:?must be set}
LITERAL=$$NOT_INTERPOLATED
```

### Encryption (AES-256-GCM)

```java
import dev.dotlyte.Encryption;

String key = Encryption.generateKey();

String encrypted = Encryption.encryptValue("s3cret", key);  // "ENC[...]"
String plain = Encryption.decryptValue(encrypted, key);       // "s3cret"
```

Values wrapped in `ENC[...]` are auto-decrypted during `load()`.

### Sensitive Value Masking

```java
config.toMapRedacted();
// {"db.host": "localhost", "db.password": "[REDACTED]"}
```

Auto-detects sensitive keys matching patterns: `password`, `secret`, `token`, `api_key`, `private_key`, etc.

### Error Types

All extend `DotlyteException` (which extends `RuntimeException`):

| Exception | Extra Properties |
|---|---|
| `DotlyteException` | `key` |
| `DotlyteException.MissingKeyException` | `sourcesChecked` |
| `DotlyteException.FileException` | `filePath` |
| `DotlyteException.ValidationException` | `violations` |
| `DotlyteException.InterpolationException` | — |
| `DotlyteException.DecryptionException` | — |

```java
try {
    config.require("missing");
} catch (DotlyteException.MissingKeyException e) {
    System.err.println(e.getKey());             // "missing"
    System.err.println(e.getSourcesChecked());  // [".env", "config.yaml"]
} catch (DotlyteException e) {
    System.err.println(e.getMessage());
}
```

## Patterns

### Spring Boot

```java
import dev.dotlyte.Dotlyte;
import dev.dotlyte.Config;
import dev.dotlyte.LoadOptions;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;

@Configuration
public class AppConfig {

    @Bean
    public Config dotlyteConfig() {
        return Dotlyte.load(LoadOptions.builder()
            .prefix("APP_")
            .env(System.getenv("SPRING_PROFILES_ACTIVE"))
            .build());
    }
}
```

### Micronaut / Quarkus

```java
Config config = Dotlyte.load(LoadOptions.builder()
    .prefix("APP_")
    .env("production")
    .build());

Object dbUrl = config.require("datasource.url");
Object dbUser = config.require("datasource.username");
```

### Validating at Startup

```java
public class Main {
    public static void main(String[] args) {
        Config config = Dotlyte.load();

        // Fail fast — validate required keys + schema at startup
        config.requireKeys("database_url", "jwt_secret", "api_key");
        config.assertValid();

        new Application(config).run();
    }
}
```

## What's New in v0.1.2

### Typed Config

```java
import dev.dotlyte.TypedConfig;

var config = TypedConfig.create(Map.of(
    "PORT", Map.of("type", "integer", "required", true, "min", 1, "max", 65535),
    "DEBUG", Map.of("type", "boolean", "default", false),
    "LOG_LEVEL", Map.of("type", "string", "enum", List.of("debug", "info", "warn", "error"))
));
```

### Boundary Config

```java
import dev.dotlyte.BoundaryConfig;

var config = BoundaryConfig.create(
    List.of("DATABASE_URL", "API_SECRET"),
    List.of("APP_NAME", "THEME"),
    null
);

var server = config.server();
var client = config.client(); // Cannot access server keys
```

```java

var enc = new Encryption("my-secret-key-32-chars-long!!!!!");

var masking = new Masking(List.of("API_KEY"));
var result = masking.mask(configMap);
```
