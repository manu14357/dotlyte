## Installation

```bash
dotnet add package Dotlyte
```

Or in your `.csproj`:

```xml
<PackageReference Include="Dotlyte" Version="0.1.2" />
```

### Requirements

- .NET **8.0+** (targets `net8.0` and `net10.0`)
- C# **12+**
- Nullable annotations enabled

## Quick Start

```csharp
using Dotlyte;

var config = DotlyteLoader.Load();

var host = config.Get("database.host", "localhost");
var port = config.Get<int>("database.port", 5432);

Console.WriteLine($"DB: {host}:{port}");
```

## API Reference

### `DotlyteLoader.Load(LoadOptions? options = null): Config`

```csharp
using Dotlyte;

// Basic
var config = DotlyteLoader.Load();

// With options
var config = DotlyteLoader.Load(new LoadOptions
{
    Files = [".env", "config.json"],
    Prefix = "APP_",
    Defaults = new Dictionary<string, object?>
    {
        ["port"] = 3000,
        ["log_level"] = "info",
    },
    Env = "production",
});
```

### `LoadOptions`

All properties use `init`-only setters:

```csharp
public sealed class LoadOptions
{
    public string[]? Files { get; init; }
    public string? Prefix { get; init; }
    public Dictionary<string, object?> Defaults { get; init; }          // = new()
    public string[]? Sources { get; init; }
    public string? Env { get; init; }
    public Dictionary<string, SchemaRule>? Schema { get; init; }
    public bool Strict { get; init; }                                    // false
    public bool InterpolateVars { get; init; }                           // true
    public Dictionary<string, object?> Overrides { get; init; }         // = new()
    public bool Debug { get; init; }                                     // false
    public bool FindUp { get; init; }                                    // false
    public string[] RootMarkers { get; init; }                           // [".git", "package.json", ...]
    public string? Cwd { get; init; }
    public bool AllowAllEnvVars { get; init; }                           // false
}
```

| Field | Type | Default | Description |
|---|---|---|---|
| `Files` | `string[]?` | `null` | Explicit config file paths |
| `Prefix` | `string?` | `null` | Env var prefix to strip |
| `Defaults` | `Dictionary<string, object?>` | `new()` | Default values (lowest priority) |
| `Sources` | `string[]?` | `null` | Source order: `"env"`, `"dotenv"`, `"json"`, `"defaults"` |
| `Env` | `string?` | `null` | Environment name |
| `Schema` | `Dictionary<string, SchemaRule>?` | `null` | Schema rules for validation |
| `Strict` | `bool` | `false` | Reject unknown keys |
| `InterpolateVars` | `bool` | `true` | Enable `${VAR}` interpolation |
| `Overrides` | `Dictionary<string, object?>` | `new()` | Highest priority values |
| `Debug` | `bool` | `false` | Debug logging |
| `FindUp` | `bool` | `false` | Walk up directories |
| `RootMarkers` | `string[]` | `[".git", ...]` | Project root markers |
| `Cwd` | `string?` | `null` | Working directory |
| `AllowAllEnvVars` | `bool` | `false` | Disable env var filtering |

Note: .NET does **not** support file watching (`Watch`/`DebounceMs`).

### `Config`

#### `Get(string key, object? defaultValue = null): object?`

```csharp
var host = config.Get("server.host", "0.0.0.0");
var missing = config.Get("nope"); // null
```

#### `Get<T>(string key, T? defaultValue = default): T?`

Typed access via `Convert.ChangeType`:

```csharp
var port = config.Get<int>("server.port", 8080);
var debug = config.Get<bool>("debug", false);
var ratio = config.Get<double>("ratio", 1.0);
```

#### `Require(string key): object`

Returns the value or throws `MissingKeyException`:

```csharp
try
{
    var dbUrl = config.Require("database_url");
}
catch (MissingKeyException ex)
{
    Console.Error.WriteLine($"Missing: {ex.Key}");
    Console.Error.WriteLine($"Checked: {string.Join(", ", ex.SourcesChecked)}");
    Environment.Exit(1);
}
```

#### `RequireKeys(params string[] keys): object[]`

Batch-require multiple keys:

```csharp
var (host, port, name) = config.RequireKeys("db.host", "db.port", "db.name")
    is [var h, var p, var n] ? (h, p, n) : throw new Exception();
```

#### `Has(string key): bool`

```csharp
if (config.Has("redis.host"))
{
    SetupRedis(config);
}
```

#### `Scope(string prefix): Config`

Create a scoped sub-config:

```csharp
var db = config.Scope("database");
db.Get("host");  // same as config.Get("database.host")
```

#### Indexer Access

```csharp
var value = config["database.host"];    // returns object?
var nested = config["database"];         // returns nested Config for dict values
```

#### Serialization & Export

```csharp
config.Keys();                 // List<string> — top-level keys
config.ToFlatKeys();           // List<string> — all keys dot-flattened
config.ToFlatDictionary();     // Dictionary<string, object?> — flat key→value
config.ToDictionary();         // Dictionary<string, object?> — shallow copy
config.ToDictionaryRedacted(); // Dictionary<string, object?> — sensitive values masked
config.ToJson();               // string — indented JSON
config.WriteTo("out.json");   // Write JSON to file (.json only)
```

#### Schema Validation

```csharp
var schema = new Dictionary<string, SchemaRule>
{
    ["port"] = new SchemaRule { Type = "integer", Required = true, Min = 1, Max = 65535 },
    ["host"] = new SchemaRule { Type = "string", Required = true },
};

List<SchemaViolation> violations = config.Validate(schema);
foreach (var v in violations)
{
    Console.WriteLine($"{v.Key}: {v.Message}");
}

config.AssertValid(schema); // throws ValidationException
```

### `SchemaRule`

```csharp
public sealed record SchemaRule
{
    public string? Type { get; init; }           // "string"|"integer"|"boolean"|"number"|"array"
    public bool Required { get; init; }
    public string? Format { get; init; }         // "email"|"url"|"uuid"|"date"|"ipv4"|"ip"|"port"
    public string? Pattern { get; init; }        // Regex pattern
    public string[]? EnumValues { get; init; }
    public double? Min { get; init; }
    public double? Max { get; init; }
    public object? DefaultValue { get; init; }
    public bool Sensitive { get; init; }
    public string? Doc { get; init; }
}
```

### Variable Interpolation

```bash
# In .env:
BASE_URL=https://api.example.com
API_ENDPOINT=${BASE_URL}/v1
GREETING=${NAME:-World}
REQUIRED=${SECRET:?must be set}
LITERAL=$$NOT_INTERPOLATED
```

Detects circular references and throws `InterpolationException`.

### Encryption (AES-256-GCM)

```csharp
using Dotlyte;

var key = Encryption.GenerateKey();  // 64-char hex string

var encrypted = Encryption.EncryptValue("s3cret", key);  // "ENC[aes-256-gcm,...]"
var plain = Encryption.DecryptValue(encrypted, key);      // "s3cret"
```

Values wrapped in `ENC[...]` are auto-decrypted during `Load()`.

Key resolution searches (in order): `DOTLYTE_KEY_{ENV}`, `DOTLYTE_KEY`, `.dotlyte-keys` file.

### Sensitive Value Masking

```csharp
var redacted = config.ToDictionaryRedacted();
// { "db.host": "localhost", "db.password": "[REDACTED]" }
```

Auto-detects keys matching: `password`, `secret`, `token`, `api_key`, `private_key`, `access_key`, `auth`, `credential`, `connection_string`, `dsn`, `encryption_key`, `signing_key`, `certificate`.

### Error Types

All inherit from `DotlyteException` (which extends `Exception`):

| Exception Class | Extra Properties |
|---|---|
| `DotlyteException` | `Key` (string?) |
| `MissingKeyException` | `SourcesChecked` (IReadOnlyList&lt;string&gt;) |
| `FileException` | `FilePath` (string?) |
| `ValidationException` | `Violations` (IReadOnlyList&lt;SchemaViolation&gt;) |
| `InterpolationException` | — |
| `DecryptionException` | — |

```csharp
using Dotlyte;

try
{
    config.Require("missing");
}
catch (MissingKeyException ex)
{
    Console.WriteLine(ex.Key);            // "missing"
    Console.WriteLine(ex.SourcesChecked); // [".env", "config.json"]
}
catch (DotlyteException ex)
{
    Console.WriteLine(ex.Message);
}
```

## Patterns

### ASP.NET Core Minimal API

```csharp
using Dotlyte;

var config = DotlyteLoader.Load(new LoadOptions
{
    Prefix = "APP_",
    Env = Environment.GetEnvironmentVariable("ASPNETCORE_ENVIRONMENT"),
});

var builder = WebApplication.CreateBuilder(args);
var app = builder.Build();

app.MapGet("/", () => "Hello, DOTLYTE!");
app.Run($"http://0.0.0.0:{config.Get<int>("port", 5000)}");
```

### ASP.NET Core with DI

```csharp
using Dotlyte;

var builder = WebApplication.CreateBuilder(args);

builder.Services.AddSingleton(_ =>
    DotlyteLoader.Load(new LoadOptions
    {
        Prefix = "APP_",
        Env = builder.Environment.EnvironmentName,
    })
);

var app = builder.Build();

app.MapGet("/config", (Config config) =>
    new { Host = config.Get("host"), Port = config.Get<int>("port") }
);

app.Run();
```

### Console Application

```csharp
using Dotlyte;

var config = DotlyteLoader.Load(new LoadOptions
{
    Files = ["appsettings.json", ".env"],
    Defaults = new Dictionary<string, object?>
    {
        ["log_level"] = "info",
        ["max_retries"] = 3,
    },
});

var logLevel = config.Get("log_level", "info");
var maxRetries = config.Get<int>("max_retries", 3);
Console.WriteLine($"Log level: {logLevel}, Max retries: {maxRetries}");
```

### xUnit Testing

```csharp
using Dotlyte;
using Xunit;

public class ConfigTests
{
    [Fact]
    public void DefaultsAreApplied()
    {
        var config = DotlyteLoader.Load(new LoadOptions
        {
            Defaults = new Dictionary<string, object?>
            {
                ["host"] = "localhost",
            },
        });

        Assert.Equal("localhost", config.Get("host"));
    }

    [Fact]
    public void RequireThrowsForMissingKey()
    {
        var config = DotlyteLoader.Load();
        Assert.Throws<MissingKeyException>(() => config.Require("nonexistent"));
    }

    [Fact]
    public void TypedGetWorks()
    {
        var config = DotlyteLoader.Load(new LoadOptions
        {
            Defaults = new Dictionary<string, object?> { ["port"] = 8080 },
        });

        Assert.Equal(8080, config.Get<int>("port"));
    }
}
```

## What's New in v0.1.2

### Typed Config

```csharp
using Dotlyte;

var config = TypedConfig.Create(new Dictionary<string, FieldDescriptor>
{
    ["PORT"] = new() { Type = "integer", Required = true, Min = 1, Max = 65535 },
    ["DEBUG"] = new() { Type = "boolean", Default = false },
    ["LOG_LEVEL"] = new() { Type = "string", Enum = ["debug", "info", "warn", "error"] },
});
```

### Boundary Config

```csharp
using Dotlyte;

var config = BoundaryConfig.Create(
    serverKeys: ["DATABASE_URL", "API_SECRET"],
    clientKeys: ["APP_NAME", "THEME"]
);

var server = config.Server();
var client = config.Client(); // Cannot access server keys
```

### Enhanced Encryption & Masking

```csharp
using Dotlyte;

var enc = new Encryption("my-secret-key-32-chars-long!!!!!");
var encrypted = enc.Encrypt("secret");
var decrypted = enc.Decrypt(encrypted);

var masking = new Masking(["API_KEY"]);
var result = masking.Mask(configDict);
```

using Dotlyte;

var ws = Workspace.Detect();
