## Installation

```bash
composer require dotlyte/dotlyte
```

### Requirements

- PHP **8.1+**
- Optional: PHP `yaml` extension for YAML support

## Quick Start

```php
<?php

use Dotlyte\Dotlyte;

require_once __DIR__ . '/vendor/autoload.php';

$config = Dotlyte::load();

echo $config->get('database.host', 'localhost'); // "localhost"
echo $config->get('database.port', 5432);        // 5432
```

## API Reference

### `Dotlyte::load(?LoadOptions $options = null): Config`

```php
use Dotlyte\Dotlyte;
use Dotlyte\LoadOptions;

// Basic
$config = Dotlyte::load();

// With options
$options = new LoadOptions();
$options->files = ['.env', 'config.yaml'];
$options->prefix = 'APP_';
$options->defaults = ['port' => 3000];
$options->env = 'production';

$config = Dotlyte::load($options);
```

### `LoadOptions`

| Field | Type | Default | Description |
|---|---|---|---|
| `files` | `?array` | `null` | Explicit config file paths |
| `prefix` | `?string` | `null` | Env var prefix to strip |
| `defaults` | `?array` | `null` | Default values (lowest priority) |
| `sources` | `?array` | `null` | Source order: `"env"`, `"dotenv"`, `"yaml"`, `"json"`, `"defaults"` |
| `env` | `?string` | `null` | Environment name |
| `schema` | `?array` | `null` | Schema rules for validation |
| `strict` | `bool` | `false` | Reject unknown keys |
| `interpolateVars` | `bool` | `true` | Enable `${VAR}` interpolation |
| `overrides` | `?array` | `null` | Highest priority values |
| `debug` | `bool` | `false` | Debug logging |
| `findUp` | `bool` | `false` | Walk up directories |
| `rootMarkers` | `?array` | `null` | Project root markers |
| `cwd` | `?string` | `null` | Working directory |
| `allowAllEnvVars` | `bool` | `false` | Disable env var filtering |

Note: PHP does **not** support file watching (`watch`/`debounceMs`).

### `Config`

#### `get(string $key, mixed $default = null): mixed`

```php
$host = $config->get('server.host', '0.0.0.0');
$port = $config->get('server.port', 8080);
$missing = $config->get('nope'); // null
```

#### `require(string $key): mixed`

Returns the value or throws `MissingKeyException`:

```php
use Dotlyte\MissingKeyException;

try {
    $dbUrl = $config->require('database_url');
} catch (MissingKeyException $e) {
    echo "Missing: " . $e->getKey();
    echo "Checked: " . implode(', ', $e->getSourcesChecked());
    exit(1);
}
```

#### `requireKeys(string ...$keys): array`

Batch-require multiple keys. Returns an array of values:

```php
[$host, $port, $name] = $config->requireKeys('db.host', 'db.port', 'db.name');
```

#### `has(string $key): bool`

```php
if ($config->has('redis.host')) {
    setupRedis($config);
}
```

#### `scope(string $prefix): Config`

Create a scoped sub-config:

```php
$db = $config->scope('database');
$db->get('host');  // same as $config->get('database.host')
```

#### Magic Property Access

```php
$config->database->host  // via __get()
isset($config->database) // via __isset()
```

#### Serialization & Export

```php
$config->keys();            // array — top-level keys
$config->toFlatKeys();      // array<string> — all flattened keys
$config->toFlatArray();     // array<string, mixed> — flat key→value
$config->toArray();         // array<string, mixed> — deep nested
$config->toArrayRedacted(); // array — sensitive values masked
$config->toJson();          // string — JSON (pretty-printed)
$config->writeTo('out.json'); // Write JSON to file
```

#### Schema Validation

```php
use Dotlyte\SchemaRule;

$schema = [
    'port' => new SchemaRule(type: 'integer', required: true, min: 1, max: 65535),
    'host' => new SchemaRule(type: 'string', required: true),
];

$violations = $config->validate($schema);
foreach ($violations as $v) {
    echo "$v->key: $v->message\n";
}

$config->assertValid($schema); // throws ValidationException
```

### Variable Interpolation

```bash
# In .env:
BASE_URL=https://api.example.com
API_ENDPOINT=${BASE_URL}/v1
GREETING=${NAME:-World}
REQUIRED=${SECRET:?must be set}
LITERAL=$$NOT_INTERPOLATED
```

### Encryption (AES-256-GCM)

```php
use Dotlyte\Encryption;

$key = Encryption::generateKey();

$encrypted = Encryption::encryptValue('s3cret', $key);  // "ENC[...]"
$plain = Encryption::decryptValue($encrypted, $key);      // "s3cret"

Encryption::encryptFile('.env', '.env.enc', $key);
Encryption::decryptFile('.env.enc', '.env.dec', $key);
```

Values wrapped in `ENC[...]` are auto-decrypted during `load()`.

### Sensitive Value Masking

```php
$config->toArrayRedacted();
// ['db.host' => 'localhost', 'db.password' => '[REDACTED]']
```

### Error Types

All extend `DotlyteException` (which extends `\RuntimeException`):

| Exception Class | Extra Properties |
|---|---|
| `DotlyteException` | `getKey()` |
| `MissingKeyException` | `getSourcesChecked()` |
| `FileException` | `getFilePath()` |
| `ValidationException` | `getViolations()` |
| `InterpolationException` | — |
| `DecryptionException` | — |

```php
use Dotlyte\DotlyteException;
use Dotlyte\MissingKeyException;

try {
    $config->require('missing');
} catch (MissingKeyException $e) {
    echo $e->getKey();             // "missing"
    echo $e->getSourcesChecked();  // [".env", "config.yaml"]
} catch (DotlyteException $e) {
    echo $e->getMessage();
}
```

## Patterns

### Laravel Service Provider

```php
<?php

namespace App\Providers;

use Dotlyte\Dotlyte;
use Dotlyte\LoadOptions;
use Illuminate\Support\ServiceProvider;

class DotlyteServiceProvider extends ServiceProvider
{
    public function register(): void
    {
        $this->app->singleton('dotlyte', function () {
            $options = new LoadOptions();
            $options->prefix = 'APP_';
            $options->env = app()->environment();
            return Dotlyte::load($options);
        });
    }
}
```

```php
// Usage anywhere in Laravel
$config = app('dotlyte');
$dbHost = $config->require('database.host');
```

### Symfony

```php
// config/services.php
use Dotlyte\Dotlyte;
use Dotlyte\LoadOptions;

$options = new LoadOptions();
$options->prefix = 'APP_';
$options->env = $_ENV['APP_ENV'] ?? 'dev';

$config = Dotlyte::load($options);

$container->setParameter('database.host', $config->require('database.host'));
$container->setParameter('database.port', $config->get('database.port', 5432));
```

### Slim Framework

```php
use Slim\Factory\AppFactory;
use Dotlyte\Dotlyte;
use Dotlyte\LoadOptions;

$options = new LoadOptions();
$options->prefix = 'APP_';
$config = Dotlyte::load($options);

$app = AppFactory::create();

$app->get('/', function ($request, $response) use ($config) {
    $response->getBody()->write("Port: " . $config->get('port', 8080));
    return $response;
});

$app->run();
```

### PHPUnit Testing

```php
use PHPUnit\Framework\TestCase;
use Dotlyte\Dotlyte;
use Dotlyte\LoadOptions;
use Dotlyte\MissingKeyException;

class ConfigTest extends TestCase
{
    public function testDefaults(): void
    {
        $options = new LoadOptions();
        $options->defaults = ['host' => 'localhost'];
        $config = Dotlyte::load($options);

        $this->assertEquals('localhost', $config->get('host'));
    }

    public function testRequireMissing(): void
    {
        $config = Dotlyte::load();
        $this->expectException(MissingKeyException::class);
        $config->require('nonexistent');
    }
}
```

## File Structure

```
langs/php/src/
├── Dotlyte.php            # Static load() entry point
├── Config.php             # Config class with all access methods
├── LoadOptions.php        # Options class
├── Loader.php             # File loading logic
├── DotlyteException.php   # All exception classes
├── Coercion.php           # Type coercion engine
├── Merger.php             # Deep merge
├── Interpolation.php      # ${VAR} interpolation
├── Encryption.php         # AES-256-GCM encrypt/decrypt
├── Validator.php          # Schema validation
├── SchemaRule.php         # Schema rule definition
├── SchemaViolation.php    # Validation violation
├── Masking.php            # Sensitive value redaction
└── Parsers/               # File format parsers
```
